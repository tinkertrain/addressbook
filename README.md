# AddressBook

## Run dev
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Go to 'http://localhost:3000'

## Run production
1. Clone the repo
2. Run `npm install`
3. Run `npm run build`
4. Files located on 'dist' folder
5. Run a web server from the dist, like `python -m SimpleHTTPServer`
